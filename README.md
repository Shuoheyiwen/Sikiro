# Introduction
本项目是我在2019年-2020年做微服务系统的例子，里面有我当时对一些组件集成与封装。
在2019年使用的.Net Core的时候还是2.2版本，后续直接升级到了.Net 5，如果有问题可以反馈给我
项目只是参考例子，并不算框架，当然我也不认为需要所谓的“微服务框架”
个人认为上面系列的博客比项目更加有价值，欢迎阅读与提出建议。
